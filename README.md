# neodgm-pro

## Install

```
npm install --save neodgm-pro
```

or

```
yarn add neodgm-pro
```

## Usage

```js
import 'neodgm-pro';
```

or

```js
require('neodgm-pro');
```

## HTML Usage

Insert following HTML in `<head>` tag.

```
<link rel="stylesheet" href="https://unpkg.com/neodgm-pro/index.css" />
```
